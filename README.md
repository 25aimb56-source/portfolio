# Portfolio Website 🚀

A modern, full-stack portfolio website showcasing my web development projects and skills.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Node.js (Express)
* **Deployment:** Vercel
* **Version Control:** Git & GitHub

## 📦 Features
* Responsive design for mobile and desktop.
* Backend server handling API requests via `server.js`.
* Continuous Deployment (CD) via GitHub Actions and Vercel.
